# Application

This example shows how to build a project that:

* Links 2 Dub Dependencies,
* Links 1 DProjectManager Dependency,
  * Which In turn links glfw.
* And includes a C++ file.

# Building

from the root directory, after building DProjectManager as noted in the root directory's README run this command:

    DProjectManager/bin/DProjectManager Examples/Application/Game/project.d

A blank window should appear.
